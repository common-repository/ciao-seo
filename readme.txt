== Description ==

Ciao SEO riporta dei semplici ma efficaci consigli SEO, in italiano, che rispettano a pieno le linee guida di Google e dei motori di ricerca. Il nome "Ciao SEO" deriva dal fatto che questo plugin si basa sul plugin "Hello Dolly".

== Installation ==

L'installazione di Ciao SEO è semplice, come dire ciao :)

### INSTALLA CIAO SEO DA WORDPRESS

1. Visita la pagina dei plugins presente nella dashboard di WordPress e seleziona ‘Aggiungi Nuovo’;
1. Cerca ‘Ciao SEO’;
1. Attiva il plugin.

### INSTALLA CIAO SEO MANUALMENTE

1. carica la cartella ‘ciao-seo’ nella directory /wp-content/plugins/;
1. Attiva Ciao SEO dalla lista dei plugin dopo aver premuto il menu ‘Plugin’ presente nella dashboard di WordPress.

### DOPO L'ATTIVAZINE

Nulla da configurare, vedrai i consigli SEO in alto la dashboard di WordPress
